return
- 함수를 빠져나감
- 값을 반환

반환형이 void인 함수여도 return은 함수가 빠져나간다는 의미도 있기떄문에 return을 사용하는 경우가 있음

지역변수의 특성
선언된 함수 내에서만 접근이 가능
함수 내에 선언된 지역변수는 해당 함수가 반환하면 소멸

static 변수의 특성
선언된 함수 내에서만 접근이 가능(지역변수 특성)
딱 1회 초기화되고 프로그램 종료시까지 메모리 공간에 존재(전역변수 특성)

register변수
지역변수에는 다음과 같이 register라는 선언을 추가할 수 있음 (cpu내에 존재하는 레지스터를 활용...)
컴파일러가 합당하다고 생각되지 않으면 선언을 하더라도 레지스터에 할당하지 않음
register int num=3;
