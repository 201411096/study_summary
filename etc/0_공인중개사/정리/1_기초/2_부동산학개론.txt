	ㄴ 법률적 개념
		ㄴ 협의의 부동산 : 민법상의 부동산 = 토지 및 그 정착물
		ㄴ 광의의 부동산 : 협의의 부동산 + 준부동산(의제부동산)
			ㄴ 준부동산
				ㄴ 등기 등록 등 공시방법을 갖춤으로서 부동산에 준하여 취급함
				ㄴ 저당권, 감정평가, 중개가 가능
				ㄴ 예시 : 자동차, 선박, 항공기, 건설기계, 공장재단, 광업재단, 입목, 어업권
				ㄴ 기계, 재단, 기타

	ㄴ 경제적 개념
		ㄴ 자산, 자본, 생산요소(생산재), 소비재, 상품
		ㄴ 산본생소상*
			ㄴ 자산 = 자본 + 부채
			ㄴ 생산요소 : 토지, 노동, 자본 (밀가루)
			ㄴ 소비재 : 주택, 상가, 농작물 (빵)
	ㄴ 기술적 개념
		ㄴ 자연, 공간, 위치, 환경
		ㄴ 공자위환*
ㄴ 복합부동산(부동산활동의 관점)

	ㄴ 법률적인 것과는 아예 상관이 없음

지목의 종류
	ㄴ 전, 답, 과수원, 대, 공장용지, 하천, 제방, 구거, 유지, 종교용지, 사적지
	ㄴ 농지 => 전, 답, 과수원

토지 용도별 분류
ㄴ 택지 : 주 상 공
ㄴ 농지 : 전 답 과
ㄴ 임지 : 용 신

토지의 분류
	ㄴ 택지 : 건축이 가능한 토지(주거용, 상업용, 공업용지)
	ㄴ 부지 : 건축이 가능한 토지(택지) + 건축이 불가능한 토지
	ㄴ 소지 : 택지가 되기 전 자연 그대로의 상태(택지개발 전, 소박한 땅, 원시적인 땅 => 소지 또는 원지)
	ㄴ 한계지 : 택지이용의 최원방권( 택지의 끝에 있는 땅 )
		ㄴ 상 > 주 > 공 > 가장 마지막 부분(한계지) > 농지
	ㄴ 후보지 : 용도전환 과정에 있는 토지(택지 농지 임지 "상호간의" 전환 / 토지 용도별 분류 중 세로)
	ㄴ 이행지 : 용도전환 과정에 있는 토지(택지 농지 임지 "내에서의" 전환 / 토지 용도별 분류 중 가로)
	ㄴ 필지 : 하나의 "지번"이 붙는 "등록 . 등기단위"이다. "법률적 개념" => 시험 시 지번이 생략되기도 함
	ㄴ 획지 : "가격수준"이 비슷한 "일단의 토지" / "경제적 개념"
	ㄴ 대지 : "좁은 통로에 의해서 도로에 접"하는 "자루형 모양"의 토지
	ㄴ 맹지 : "도로에 어떤 접속면도 가지지 못하는" 토지 / 건축법상 "건축이 불가능"
	ㄴ 나지 : 토지에 건물 및 기타 정착물이 없고, 사법상의 권리가 설정되어 있지 않은 토지(공법상의 제약은 존재함 => 우리나라의 모든 토지는 공법상의 제약이 있기 때문)
		ㄴ 나(벗을 나)
	ㄴ 건부지 : 건축물이 이미 있는 토지
		ㄴ 건부감가(원칙)
			ㄴ 기본적으로 토지의 가격만 비교할 경우(나지의 가치가 건부지보다 높음)
	ㄴ 공지 : 건축법에 의한 건폐율, 용적률 등의 "공적제한"으로, 한 필지 내에서 건축하지 않고 비워둔 토지
		ㄴ 건부지 중 비어있는 땅 -> 공지
		ㄴ 건부지 = 공지 + 건물 바로 밑의 땅
	ㄴ 공한지 : "도시토지" / 지가상승을 기대하고 투기목적으로 "방치"하고 있는 토지
	ㄴ 휴한지 : 쉬고 있는 토지 (첫글자가 메인 "휴" -> 쉬고있는)
		ㄴ 농지의 비옥도 혹은 지력의 회복을 위해 정상적으로 쉬고 있음
	ㄴ 유휴지 : 놀고 있는 토지 (첫글자가 메인 "유" -> 놀고있는)
	ㄴ 법지 : 법적 소유권은 있으나 이용가치 X
		ㄴ 경사진 땅...
	ㄴ 빈지 : 법적 소유권은 없으나 이용가치 O / 바다와 육지 사이의 토지 (바다**)
		ㄴ 경사진 땅...(바닷가)
	ㄴ 선하지 : 고압전선 아래의 토지
	ㄴ 포락지 : 지적에 등록되어 있으나 "물에 침식되어 수면 밑으로 잠긴 토지" (하천**)
	ㄴ 일단지 : 용도상 불가분의 관계에 있는 2필지 이상의 일단의 토지
		ㄴ 토지의 분류에 "일단"이 나오는 경우
			ㄴ 두필지 이상의 불가분 토지 : 일단지
			ㄴ 가격수준이 비슷한 일단의 토지 : 획지
	ㄴ 표준지**
		ㄴ "지가의 공시"를 위해 "선정한 토지"
		ㄴ 표준지가공시**
	ㄴ 표본지
		ㄴ 지가변동을 측정하기 위한 토지

표준산업분류에 따른 부동산의 분류 => 직전시험에 나와서 안 나올 가능성이 높긴 함
ㄴ 대분류 - 소분류 - 세분류 - 세세분류
ㄴ 부동산업
	ㄴ 부동산임대 및 공급업 => "임개공"
		ㄴ 부동산임대업
			ㄴ 주거 / 비주거 / 기타
		ㄴ 부동산개발 및 공급업
			ㄴ 주거 / 비주거 / 기타
	ㄴ 부동산 관련 서비스업
		ㄴ 부동산관리업
			ㄴ 주거 / 비주거 => 기타가 없음***
		ㄴ 부동산중개, 자문 및 감정평가업 => "중자감"
			ㄴ 중개 및 대리, 투자자문, 감정평가
			ㄴ 투자X, 금융X

	ㄴ 부 -> 정 : 비율로는 변경 불가능

화폐의 시간가치 계수 정리
ㄴ 1. 일시불의 내가계수	= 복리증가율	= (1+r)^n
ㄴ 2. 일시불의 현가계수	= 복리현가율	= 1 / (1+r)^n
ㄴ 3. 연금의 내가계수	= 복리연금증가율	= ( (1+r)^n - 1 )/r
ㄴ 4. 연금의 현가계수	= 복리연금현가율	= ( 1 - (1+r)^n )/r
ㄴ 5. 감채기금 계수		= 상환기금률	= r/( (1+r)^n - 1 )
ㄴ 6. 저당상수		= 연부상환율	= r/( 1 - (1+r)^n )

현가 = 현재가치

예제8
A는 연소득 5000만원, 시장가치 3억의 주택, 담보로 5000만원 대출
추가로 대출가능한 금액은?
연간 저당상수 : 0.1
LTV : 50%이하	/ DTI : 40%이하
ㄴ LTV <-> DTI 중에서 작은 숫자만큼 대출을 받을 수 있음
ㄴ LTV = 부채 / 부동산가치(3억) = 50 => 1억 5천만원
ㄴ DTI = 부채 * 저당상수 / 연간소득(5000만) = 0.4 => 부채 * 저당상수(0.1) = 5000만 * 0.4 => 2억
ㄴ LTV DTI 중에서 값이 작은 1억 5천만원까지 대출이 가능하기 때문에, 1억을 더 대출 받을 수 있다.

예제11. 순현가(NPV)와 내부수익률(IRR)은?
초기현금지출 5000원, 말기현금유입 6000원
할인율 20% 사업기간 1년, 사업초기만 현금지출

수입내가 = 6000원
지출현가 = 5000원
수입현가 = 6000 * 1 / (1+r)^n = 6000 / 1.2

NPV = 수입현가 - 지출현가 = 5000 - 5000 = 0
IRR = 차이값 / 투자액 = (6000 - 5000) / 5000 = 20%
PI = 수입현가 / 지출현가 = 5000 / 5000 =1

원가방식

원가법
ㄴ 공식 : 적산가액 = 제조달원가 - 감가누계액(감가수정)
ㄴ 제조달원가 : 기준시점에서 새롭게 신축하는 데 소요되는 신축비
	ㄴ 도급건설에 준하여 재조달원가를 산성
		ㄴ 도급건설 = 부동산 소유자가 업자에게 맡김 => 수급인(건설을 맡은 건설업자)의 적정이윤도 포함됨
	ㄴ 건물의 제조달원가 = 표준적 건설비용 + 통상의 부대비용
		ㄴ 표준적 건설비용 : 직접공사비, 간접공사비, 수급인의 적정이윤
	ㄴ 산정방법
		ㄴ 1. 지수법	: 신축공사비 * 기준시점 지수 / 건축시점 지수 (현재공사비 * 현재물가/과거물가?)
		ㄴ 2. 변동률적용법	: 신축공사비 * (1+r)^n
ㄴ 감가수정
	ㄴ 직접법
		ㄴ 정액법
			ㄴ 매년 감가액	= 감가총액(제조달원가-잔존가치) / 경제적 내용연수
			ㄴ 감가누계액	= 매년감가액 * 경과연수
			ㄴ 적산가액	= 제조달원가 - 감가누계액
		ㄴ 정률법

내용연수 : 전체수명
경과연수 : 현재수명
잔존가치 : 내용연수가 지나도 떨어지지 않는 가치
ㄴ 사례
	ㄴ 내용연수 20년
	ㄴ 잔존가치 10%
	ㄴ 제조달원가 : 2억
	ㄴ => 잔존가치 : 2천 (20년이 지나도 떨어지지 않음)
	ㄴ => 감가총액 : 1억 8천
	ㄴ => 매년누계액 : 1억 8천/20 = 900만(정액법에서는 항상 900씩 까임)
	ㄴ 만약에 잔존가치가 주어지지 않는다면 ?? => 제조달원가 = 감가총액**
감가총액 = 제조달원가 - 잔존가치

비교방식
ㄴ 거래사례비교법

비준가액 = 거래사례 * (시점보정 * 시점수정 * 가치형성요인비교)
토지가액 = 비교표준지공시지가 * (시점보정 * 지역요인 * 개별요인 * 그 밖의 요인 보정)

사정보정률 = 대상부동산 / 사례부동산 ( 대상받고싶어서 위로 .. / 사례.. 아래..)

시점수정방법
ㄴ 지수법 = 기준시점의 가격지수 / 거래시점의 가격지수
ㄴ 변동률 = 시점수정치 = (1+r)^n

수익환원법

수익가액 = 순수익(순영업소득) / 환원이율

순영업소득 = 가공기유영"순"부세영세
	ㄴ 순까지만

영업경비 = 유지수선비, 화재보험료, 재산세

환원이율 = 저당상수 * 부채감당률 * 대부비율
ㄴ 저부대
ㄴ 부채감당률 = 순영업소득/부채서비스액
ㄴ 순수익(순영업소득)/부동산의 가격 	********************************
ㄴ 장래에 기대되는 순수익을 현재가치로 환원하여 대상부동산의 가격을 구하는 데 사용

부채감당률 = 순영업소득/부채서비스액
