물권과 채권의 우선순위
ㄴ 물권은 서로 간에 우선순위가 존재하며, 채권보다는 항상 먼저 받아감
ㄴ 채권은 서로 간에 우선순위와 상관없이 평등하게 분배되며, 물권보다 후순위임

우선변제권
ㄴ 후순위보다 먼저 받아감, 일반채권자보다 먼저 받아감, 가장 먼저 받아간다는 이야기는 아님
ㄴ 예시 : 경매 시 돈을 먼저 받아감(채권보다)


법률사실 -> 법률요건 -> 법률효과

법률행위 : "의사표시"를 "필수요소"로 하는 법률요건

	ㄴ 의사표시 + a -> 법률행위는 의사표시만으로 구성된것은 아님
		ㄴ ex) 청혼 + 승낙 -> 혼인신고 => 의사표시 + a -> 결혼

의사표시 : 법률사실

권리변동의 원인

	ㄴ 법률규정 : ex) 상속

법률행위의 목적

ㄴ 가능 : 실현의 가능 여부는 물리적 기준이 아닌, 사회적통념에 의하여 정해짐
ㄴ 적법

목적의 실현 가능성 ( 법률행위의 목적_2 - 가능)
ㄴ 실현의 가능 여부는 물리적 기준이 아닌, 사회적통념에 의하여 정해짐
ㄴ 원시적 불능과 후발적 불능
	ㄴ 원시적불능(무효)
	ㄴ 후발적불능(유효)
		ㄴ 채무자의 잘못이 있다면 -> 채무불이행 => 상대방은 계약 해제 및 "손해배상 청구"가 가능
ㄴ 불능 여부는 법률행위 성립시를 기준으로 판단

법률용어 : 적극 <-> 소극 (적극 : 인정, 소극 : 부정)

목적의 적법성 ( 법률행위의 목적_3 - 적법)
법률규정
ㄴ 강행규정
	ㄴ 효력규정 : 무효
	ㄴ 단속규정 : 유효(개인간의 거래는...) / 하지만 제재는 받음
ㄴ 임의규정 : 계약자유의 원칙

위조등기시 "실체관계가 있어야만" 등기가 유효함(형사상 처벌을 받음)

계약 무효, 취소 => 부당이득
	ㄴ 부당이득이란 법률상 원인없이 취득(계약이 사라져서... 원인이 사라짐...)

목적의 사회적타당성 ( 법률행위의 목적_4 - 사회적타당성) ***
선량한 풍속, 기타 사회... 위반한 내용은 무효로 한다.
ㄴ 인륜에 반한 ..
	ㄴ ex) 첩계약
	ㄴ 예외 : 첩계약을 종료하면서 생활비등은 유효
ㄴ 정의관념에 반한 ..
	ㄴ ex) 범죄
ㄴ 개인의 자유를 극도로 제한
	ㄴ ex) 평생시리즈 (평생 일하거나.., 평생 독신...)
ㄴ 생존의 기초를 처분 ..
ㄴ 사행성이 심하면 ..
	ㄴ ex) 도박

사회질서 위반인 행위의 효과
ㄴ 이행전 이면 이행필요 X, 이미 이행된것은 부당이득반환의 대상
ㄴ 반사회적행위에 의해 무효가 되면 부당이득반환의 특칙으로서 "불법웝인급여"가 적용된다.

법조문 tip
~다. 그러나

ㄴ ① => 1항
ㄴ 1.  => 1호
ㄴ 가. => 목

ㄴ 104조 당사자의 궁박 . 경솔 또는 무경험일 경우 공정을 잃은 법률행위 => 무효
ㄴ 103조 중 한가지의 예시
ㄴ "객관적 요건"과 "주관적 요건"이 필요함
	ㄴ 객관적 요건 : 현저한 불균형	=> 법률행위시를 기준으로 함
	ㄴ 주관적 요건 : 궁박, 경솔, 무경험
		ㄴ 궁박 : 경제적, 정신적, 신체적인 곤궁함
		ㄴ 무경험 : 특정거래영역 X, 일반거래상O
			ㄴ ex) 부동산 거래를 처음해본건 상관없음(이전에 경제활동을 했었으니까)
			ㄴ ex2)이전에 부동산 거래를 했었던 80세 할머니(문맹) => 일반적인 거래 활동이 불가능 => 무경험자로 침
	ㄴ 이러한 객관적 요건과 주관적 요건은 추정되지 않음 -> 무효주장자가 증명해야함
	ㄴ 103조의 일부로서 불법원인급여가 적용이 됨(오히려 이득을 볼 수도 있음)
	ㄴ 절대적 무효 => 선의의 제3자는 대항할 수 없음
원칙과 예외가 있다면 => 예외주장자가 입증
본문과 단서가 있다면 => 단서주장자가 입증
