2. 확정, 가능, 적법, 사회적타당성
ㄴ 가능
	ㄴ 원시적불능 : 무효

3. 의사와 표시의 일치 / 의사형성에 하자가 없을 것
ㄴ 의사와 표시의 불일치
	ㄴ 비진의 표시
	ㄴ 통정허위표시
	ㄴ 착오
ㄴ 의사형성에 하자가 있는 경우
	ㄴ 사기, 강박

=> 비진의표시, 통정허위표시, 착오, 사기/강박은 기본적으로 선의의 제3자를 보호함
ㄴ 선의는 추정되며, 악의를 주장하는자가 입증해야한다.

통정허위표시
ㄴ 의사표시가 있을것
ㄴ 의사와 표시가 불일치 할 것
ㄴ 표의자가 의사와 표시의 불일치를 알고 할 것
ㄴ 상대방과 통정(합의)할 것***
ㄴ 허위 표시의 이유나 동기는 묻지 않는다.
ㄴ 통정허위표시 사례
	ㄴ 경매로 넘어갈 건물을 다른사람과 합의하여 가장매매함 => 기본적으로 무효임
		ㄴ 가장매매자체가 불법은 아니며, 불법원인급여에 해당하지 않음(가장매매를 했어도 원래 소유자의 것임)

엄폐물 이론
ㄴ 비진의표시, 통정허위표시에 대한 내용을 다 알고 있어도.. 선의의 제3자에게 구매했을 경우 권리자에게 구매한것이므로 유효하다.

특정승계 : 매매, 교환
포괄승계 : 상속, 포괄유증, 합병
ㄴ 포괄승계를 받은 자는 제3자라고 볼 수 없다.***

유증
ㄴ 포괄유증

착오로 인한 의사표시
ㄴ "내용"의 "중요부분"에 착오가 있는 때에는 취소할 수 있다. 하지만 그 착오가 표의자의 "중대한 과실"로 인한 때에는 취소하지 못한다.
ㄴ 내용
	ㄴ 동기의 착오는 원칙적으로 되지 않음
ㄴ 중요부분
	ㄴ 객관적으로 볼때도, 주관적으로 볼때도 중요해야 함
		ㄴ 객관적 : 일반인
		ㄴ 주관적 : 표의자
ㄴ 중대한과실이 없을 것
	ㄴ (경)과실 : 기본적으로 취소할 수 있음
	ㄴ 중과실 : 주의/의무를 "현저히" 게을리 한 경우
		ㄴ 중과실이 있었음에도 ... => 상대방이 이를 알고 "이용"***한 경우에는 중요한 과실이 있었어도 취소가 가능함
	ㄴ 공장사장 판례 ... => 중과실
	ㄴ 고려청자 판례 ... => 중과실이 없음(자신을 과신한 나머지...)
ㄴ 중요부분의 경우는 표의자가 입증해야하며 / 중대한과실이 있었을 것이라는 것은 상대방(취소를 막고 싶은 사람)이 입증해야함
ㄴ 사례
	ㄴ 취소가 되는 경우
		ㄴ 1. 토지의 현황과 경계의 착오
			ㄴ 현황 : 하천부지 판례... => 기본적으로 "물리적인 현황임"

해제와 취소
ㄴ 공통점
	ㄴ 단독행위
	ㄴ 소급효과
	ㄴ 의사표시(권리의 행사가 있어야함, 취소사유, 해제사유가 있다고 해서 저절로 취소 혹은 해제가 되지 않음)
ㄴ 차이점
	ㄴ 해제
		ㄴ  "계약"만을 해제
		ㄴ 원상회복
		ㄴ 법정해제 : 채무불이행(+손해배상)
		ㄴ 약정해제 : 해제사유가 발생할 경우 일방적으로(채무불이행은 아니기 때문에 손해배상은 하지않음)
			ㄴ 이자는 가산한다?..
		ㄴ 합의해제 : 서로 해제함..(민법규정이 적용되지 않음 => 손해배상도 하지 않음, 이자도 가산하지 않음)****************************************************
	ㄴ 취소
		ㄴ "계약, 단독행위"등을 취소
		ㄴ 부당이득


사례
ㄴ 매매계약
ㄴ 무효, 취소
	ㄴ 부당이득
	ㄴ 선의 => 과실취득 O
	ㄴ 선의 => 이자가산 X
ㄴ 해제
	ㄴ 원상회복
	ㄴ 과실취득 X
	ㄴ 이자가산 O

합의해제(=해제계약)*********************
ㄴ 1. 민법규정 적용X
	ㄴ 손해배상X
	ㄴ 이자가산X
ㄴ 2. 소유권당연복귀 => 등기말소청구 : 소유권에 의한 물권적청구(소멸시효 X)************************
ㄴ 3. 제3자는 보호**

해제권의 발생
ㄴ 1. 약정해재권
	ㄴ 채무불이행이 아니므로 손해배상청구 X
	ㄴ 이자는 가산한다.
ㄴ 2. 법정해제권
	ㄴ 채무불이행 시 해제할 수 있음
		ㄴ 이행지체
			ㄴ 상당한 기간을 정하여 최고
			ㄴ 기간이 미달하는 최고 : 유효 / 상당한 기간이 경과해야 해제
			ㄴ 기간지정을 하지 않아도 됨(최고만 하면 됨)
		ㄴ 이행불능
		ㄴ 불완전이행
			ㄴ 추완이 가능한 경우
				ㄴ 상당기간 최고
			ㄴ 추완이 불가능한 경우
				ㄴ 즉시해제
		ㄴ 사정변경
		ㄴ 부수적 의무의 불이행으로 인한 해제권
			ㄴ 해제는 할 수 없음(주된 의무가 아니라면..)

최고 : 재촉하고 알리다

최고를 하지 않아도 되는 것
ㄴ 정기행위 => 의사표시가 필요
	ㄴ 그때가 아니라면 목적달성이 되지 않음
	ㄴ ex) 돌잔치 떡, 결혼식용 요리 주문
ㄴ 거절의사표시
	ㄴ 거절의사를 철회하면 최고를 해야함

이행불능
	ㄴ 최고가 필요없음
	ㄴ 이행시기까지 기다릴 필요가 없음
	ㄴ 이행제공이 필요없음


해제권의 행사
ㄴ 해제권의 불가분성
	ㄴ 계약의 해지나 해제는 전원으로부터 전원에 대하여 한다.
		ㄴ 사례
			ㄴ 갑, 을, 병이 같이 하나의 부동산에 대해서 거래할 경우... 셋 중 하나만 문제가 있어도 전부에 대하여 해제..
		ㄴ 대표자를 두었다면 해제되지 않음..

해제의 효과*****
ㄴ 원상회복의 의무가 있다.
ㄴ 이자를 가산해야 한다.(그 받은날로부터..)
ㄴ 손해배상은 할 수 없다.
ㄴ 제3자의 권리를 해하지 못한다.
	ㄴ 해제 되기 전 취득 : 선악불문
	ㄴ 해제 이후 취득 : 선의
ㄴ 비통착사에서는 채권도 보호해주지만, 해제에서는 완전한 권리만 보호해줌
