계약의 과정
1. 청약(약속을 청함)
2. 승낙
3_1. 청약+승낙 => 계약 성립
3_2. 청약+승낙x=> 계약 불성립, 계약 부존재
4. 계약의 유효
	ㄴ 능력
	ㄴ 확정성, 가능성, 적법성, 사회질서에 부합
	ㄴ 의사 = 표시
		의사와 표시가 일치되지 않을 경우 -> 취소
5. 계약의 이행
	ㄴ 부동산을 안 넘겨주거나, 대금을 안 넘겨주거나..	-> 채무불이행
	ㄴ 채무불이행 -> 계약의 해제(소멸)
		ㄴ 위험부담(건물이 소멸했거나, 자연재해)
6. 물권의 변동
	ㄴ 물권 : 물건에 대한 권리
	ㄴ 물권이 변동되기 위한 변동... (기본적으로 등기가 되었을 때)
	ㄴ 물권적 청구권(자신의 물권을 보호하기 위한...)

민법총칙의 순서
1장. 법률행위 총칙
2장. 의사표시
3장. 법률행위의 대리
4장. 법률행위의 무효와 취소
5장. 법률행위의 부관

법률행위란
	ㄴ 의사표시를 필수로 함, 계약을 했다고 생각해도 무방함
	ㄴ 법률행위를 하면 그에 따라 법률효과가 발생하는데 그 법률행위의 효과를 만들어내는 것이 의사표시이다.
	ㄴ 법률행위는 의사표시를 필수요소로 하는 법률요건(하지만 반드시 동일한 것은 아님)
		ㄴ ex) 혼인의 경우 의사표시가 필수이나 신고의 절차도 필요함(의사표시 != 법률행위)

법률행위의 종류
1. 단독행위 : 행위자의 1개의 의사표시만으로 성립하는 법률행위
	ㄴ 상대방 없는 단독행위
		ㄴ 유언(유증), 재단법인의 설립행위, 소유권 포기		-> 유재포
			ㄴ 유증 : 유언으로 증여한다.
			ㄴ 법인 : 단체가 사람이 될 수 있음
	ㄴ 상대방 있는 단독행위
		ㄴ 취소, 철회, 동의, 추인, 해제, 해지, 채무면제, 상계 - 일단 넘어가기
		ㄴ 상대방의 의사에 관계없이 일방적으로 그 효력이 발생하는 단독행위
	ㄴ 누군가 계약과정에서 사기를 당했다면, 사기를 당한 사람은 취소를 할 수 있음
		ㄴ 상대방의 의사가 필요없음(형성권)
		ㄴ 사기나 강박에 의한 의사표시는 취소가 가능함
	ㄴ 단독행위는 법률이 정할 때 할 수 있음
2. 계약 : 청약과 승낙에 의한 합의가 필요함
	ㄴ 매매, 교환, 임대차
	ㄴ 계약자유의 원칙상 15가지의 계약외에 새로운 계약을 만들 수 있음(무명계약, 비전형계약 <-> 유명계약, 전형계약) 

법률행위의 성립요건과 유효요건
	ㄴ 성립이 되었다고 해서 유효한건 아님
	ㄴ 성립이 되지 않았다면 불성립(무효x) -> 무효와 취소란 성립요건을 갖춘 이후에 따질 수 있음

법률행위의 성립요건
	ㄴ 일반적으로 필요한 요건 -> 당사자, 목적, 의사표시
법률행위의 유효요건(효력요건)
	ㄴ 당사자가 능력을 가지고 있을 것
		ㄴ 제한능력자(미성년자, 피한정후견인, 피성년후견인) -> 취소
			ㄴ 치산 : 재산을 다루다
			ㄴ 한정치산자 : 재산 다루는 게 한정적	-> 피한정후견인
			ㄴ 금치산자 : 재산 다루는 게 금지	-> 피성년후견인(상태가 피한정후견인보다 안좋음)
		ㄴ 의사무능력자(광인, 술에 만취한 사람, 유아) -> 무효
	ㄴ 법률행위의 목적이 확적성, 가능성, 적법성, 사회적 타당성이 있을 것	-> 확가적사
	ㄴ 의사표시에 관하여 의사와 표시가 일치하고, 의사표시에 하자가 없을 것
		ㄴ 비정상적인 의사표시는 무효 또는 취소가 될 수 있다.
			ㄴ 비진의표시, 허위표시, 착오에 의한 의사표시, 사기-강박에 의한 의사표시
무효와 취소의 차이점
	ㄴ 무효 : 일단 계약이 성립은 되었으나, 애초에 처음부터 효력이 없음
	ㄴ 취소 : 취소를 안 했다면 불확정적 유효, 취소를 하게 된다면, 처음으로 돌아가서 효력이 없어짐(소급 효과)
		ㄴ 취소를 했다면 무효와 동일하다고 볼 수 있음
		ㄴ 소급효과 : 처음으로 거슬러가서..

법률행위의 목적
1. 목적의 확정성
법률행위의 목적은 확정할 수 있어야하며, 법률행위의 내용을 실현할 당시(이행기)까지 확정할 수 있으면 된다.
	ㄴ 계약 성립 당시에 확정되어야 한다 -> (X)
2. 목적의 가능성
목적은 실현이 가능한 것이여야 한다. 가능인지, 불능인지는 사회통념에 의해서 정해진다(물리적 기준X)
3. 목적의 적법성
법규
	ㄴ 강행규정
		ㄴ 기본적으로 강행규정이라고 이야기하면 효력규정임(민법에서는 100%)
		ㄴ 효력규정 : 사법상 효력을 무효화시키는 규정
			ㄴ 위반 시 무효
		ㄴ 단속규정 : 사법상 효과에는 영향을 미치지 않음
			ㄴ 위반 시 처벌하지만 사법적으로는 유효함
			ㄴ 불허가음식점 : 처벌받지만 밥값은 내야함
	ㄴ 임의규정 : 당사자의 의사가 우선
		ㄴ ex) 임대인 <-> 임차인 : 기본적으로 매월말에 주는 것이 규정이지만 ... -> 당사자의 의사에 따라 변경 가능
4. 사회적 타당성
반사회적 법률행위 : 선량한 풍속 기타 사회질서에 위반한 사항을 내용으로 하는 법률행위는 무효로 한다.
	ㄴ 윤리질서에 반하는 행위
		ㄴ ex) 첩계약 (but 첩관계를 그만두는 것을 조건으로 금전을 지급하는 계약등은 유효
	ㄴ 정의관념에 반하는 행위
		ㄴ ex) 범죄 ...
	ㄴ 개인의 자유를 심하게 제한하는 행위
		ㄴ ex) 평생 결혼x, 이혼x, ..
	ㄴ 생존의 기초를 처분하는 행위
		ㄴ ex) 전 재산 양도..
	ㄴ 사행성이 심한 행위
		ㄴ ex) 도박..
반사회적 법률행위의 효과는 무효이다.
	ㄴ 원칙적으로 무효인 법률행위는 이행 전이면 이행할 필요가 없음, 이미 이행된 것은 부당이득반환의 대상,
	   다만 법률행위가 제103조에 반하여 무효가 되면 부당이득반환의 특칙으로서 불법원인급여가 적용
	ㄴ 불법원인급여 -> 반사회적으로 준 것이라면 돌려받을 수 없음
	ㄴ ex) 계약시 소유권이 넘어가고 대금이 넘어가는 것은 계약 때문임 -> 계약 취소 시에는 두가지가 부당이득이 되서, 되돌려줘야함
	ㄴ ex) 첩계약시 넘겨준 건물이 있다면 무효가 되서 부당이득의 대상이 되지만, 불법원인급여가 적용되서 돌려받을 수 없음
