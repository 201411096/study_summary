법률행위란
	ㄴ 의사표시를 필수로 함, 계약을 했다고 생각해도 무방함
	ㄴ 법률행위를 하면 그에 따라 법률효과가 발생하는데 그 법률행위의 효과를 만들어내는 것이 의사표시이다.
	ㄴ 법률행위는 의사표시를 필수요소로 하는 법률요건(하지만 반드시 동일한 것은 아님)
		ㄴ ex) 혼인의 경우 의사표시가 필수이나 신고의 절차도 필요함(의사표시 != 법률행위)

법률행위의 종류
1. 단독행위 : 행위자의 1개의 의사표시만으로 성립하는 법률행위
	ㄴ 상대방 없는 단독행위
		ㄴ 유언(유증), 재단법인의 설립행위, 소유권 포기		-> 유재포
			ㄴ 유증 : 유언으로 증여한다.
			ㄴ 법인 : 단체가 사람이 될 수 있음
	ㄴ 상대방 있는 단독행위
		ㄴ 취소, 철회, 동의, 추인, 해제, 해지, 채무면제, 상계 - 일단 넘어가기
		ㄴ 상대방의 의사에 관계없이 일방적으로 그 효력이 발생하는 단독행위
	ㄴ 누군가 계약과정에서 사기를 당했다면, 사기를 당한 사람은 취소를 할 수 있음
		ㄴ 상대방의 의사가 필요없음(형성권)
		ㄴ 사기나 강박에 의한 의사표시는 취소가 가능함
	ㄴ 단독행위는 법률이 정할 때 할 수 있음
2. 계약 : 청약과 승낙에 의한 합의가 필요함
	ㄴ 매매, 교환, 임대차
	ㄴ 계약자유의 원칙상 15가지의 계약외에 새로운 계약을 만들 수 있음(무명계약, 비전형계약 <-> 유명계약, 전형계약) 

법률행위의 성립요건과 유효요건
	ㄴ 성립이 되었다고 해서 유효한건 아님
	ㄴ 성립이 되지 않았다면 불성립(무효x) -> 무효와 취소란 성립요건을 갖춘 이후에 따질 수 있음

법률행위의 성립요건
	ㄴ 일반적으로 필요한 요건 -> 당사자, 목적, 의사표시
법률행위의 유효요건(효력요건)
	ㄴ 당사자가 능력을 가지고 있을 것
		ㄴ 제한능력자(미성년자, 피한정후견인, 피성년후견인) -> 취소
			ㄴ 치산 : 재산을 다루다
			ㄴ 한정치산자 : 재산 다루는 게 한정적	-> 피한정후견인
			ㄴ 금치산자 : 재산 다루는 게 금지	-> 피성년후견인(상태가 피한정후견인보다 안좋음)
		ㄴ 의사무능력자(광인, 술에 만취한 사람, 유아) -> 무효
	ㄴ 법률행위의 목적이 확적성, 가능성, 적법성, 사회적 타당성이 있을 것	-> 확가적사
	ㄴ 의사표시에 관하여 의사와 표시가 일치하고, 의사표시에 하자가 없을 것
		ㄴ 비정상적인 의사표시는 무효 또는 취소가 될 수 있다.
			ㄴ 비진의표시, 허위표시, 착오에 의한 의사표시, 사기-강박에 의한 의사표시
무효와 취소의 차이점
	ㄴ 무효 : 일단 계약이 성립은 되었으나, 애초에 처음부터 효력이 없음
	ㄴ 취소 : 취소를 안 했다면 불확정적 유효, 취소를 하게 된다면, 처음으로 돌아가서 효력이 없어짐(소급 효과)
		ㄴ 취소를 했다면 무효와 동일하다고 볼 수 있음
		ㄴ 소급효과 : 처음으로 거슬러가서..

법률행위의 목적
1. 목적의 확정성
법률행위의 목적은 확정할 수 있어야하며, 법률행위의 내용을 실현할 당시(이행기)까지 확정할 수 있으면 된다.
	ㄴ 계약 성립 당시에 확정되어야 한다 -> (X)
2. 목적의 가능성
목적은 실현이 가능한 것이여야 한다. 가능인지, 불능인지는 사회통념에 의해서 정해진다(물리적 기준X)
3. 목적의 적법성
법규
	ㄴ 강행규정
		ㄴ 기본적으로 강행규정이라고 이야기하면 효력규정임(민법에서는 100%)
		ㄴ 효력규정 : 사법상 효력을 무효화시키는 규정
			ㄴ 위반 시 무효
		ㄴ 단속규정 : 사법상 효과에는 영향을 미치지 않음
			ㄴ 위반 시 처벌하지만 사법적으로는 유효함
			ㄴ 불허가음식점 : 처벌받지만 밥값은 내야함
	ㄴ 임의규정 : 당사자의 의사가 우선
		ㄴ ex) 임대인 <-> 임차인 : 기본적으로 매월말에 주는 것이 규정이지만 ... -> 당사자의 의사에 따라 변경 가능
4. 사회적 타당성
반사회적 법률행위 : 선량한 풍속 기타 사회질서에 위반한 사항을 내용으로 하는 법률행위는 무효로 한다.
	ㄴ 윤리질서에 반하는 행위
		ㄴ ex) 첩계약 (but 첩관계를 그만두는 것을 조건으로 금전을 지급하는 계약등은 유효
	ㄴ 정의관념에 반하는 행위
		ㄴ ex) 범죄 ...
	ㄴ 개인의 자유를 심하게 제한하는 행위
		ㄴ ex) 평생 결혼x, 이혼x, ..
	ㄴ 생존의 기초를 처분하는 행위
		ㄴ ex) 전 재산 양도..
	ㄴ 사행성이 심한 행위
		ㄴ ex) 도박..
반사회적 법률행위의 효과는 무효이다.
	ㄴ 원칙적으로 무효인 법률행위는 이행 전이면 이행할 필요가 없음, 이미 이행된 것은 부당이득반환의 대상,
	   다만 법률행위가 제103조에 반하여 무효가 되면 부당이득반환의 특칙으로서 불법원인급여가 적용
	ㄴ 불법원인급여 -> 반사회적으로 준 것이라면 돌려받을 수 없음
	ㄴ ex) 계약시 소유권이 넘어가고 대금이 넘어가는 것은 계약 때문임 -> 계약 취소 시에는 두가지가 부당이득이 되서, 되돌려줘야함
	ㄴ ex) 첩계약시 넘겨준 건물이 있다면 무효가 되서 부당이득의 대상이 되지만, 불법원인급여가 적용되서 돌려받을 수 없음

	ㄴ 강박

비진의표시
제 107조[진의가 아닌 의사표시] 
1. 의사표시는 표의자가 진의 아님을 알고 한 것이더라도 그 효력이 있다.
그러나 상대방이 표의자의 진의 아님을 알았거나 이를 알 수 있었을 경우에는 무효로 한다.(단서조항)
	ㄴ 표의자(의사를 표시한 자) : 사직서를 제출했으나, 사직할 마음은 없었음 -> 진의가 아님을 알고 한 것임(자신의 진의를 모르고 했다면 착오)
	ㄴ 알 수 있었을 경우 -> 과실로 알지못한 경우
	ㄴ 비진의표시가 유효가되려면 선의이면서 무과실이여야 함
2. 전항의 의사표시의 무효는 선의의 제3자에게 대항하지 못한다.

진의 : 비진의표시에서 진의란 진심으로 마음속에서 원하는 사항을 의미하는 것이 아니라 특정한 의사표시를 하고자 하는 생각을 의미한다.
	ㄴ 원하느냐의 문제가 아님
	ㄴ 협박을 받아서 재산을 증여한 경우 -> 원하지는 않았지만 재산을 증여한다는 생각은 있었음 -> 진의 -> 비진의표시가 아님
	ㄴ 아파트를 모으는 게 취미 -> 재산폭탄 -> 어쩔 수 없이 매매계약을 함 -> 진의 -> 원하지는 않았지만 판다는 생각이 있었음

법에서의 선의/악의, 무과실/과실
선의 : 어떤 사실을 모르고 있음
악의 : 어떤 사실을 알고 있음
무과실 : 어떤 사실을 알 수 없었음
유과실 : 어떤 사실을 알 수 있었음

비진의표시의 요건
1. 의사표시가 있어야 함
2. 의사와 표시가 불일치하여야 함
3. 표의자 스스로 의사 표시의 불일치를 알아야만 함

효과
1. 원칙(유효) : 비진의표시는 표시된 대로 그 효과가 발생
2. 예외(무효) : 상대방의 표의자의 진의 아님을 알았거나, 이를 알 수 있었을 경우에(악의가 있거나, 과실인 경우에)
	상대방의 악의 또는 과실 유무에 대한 입증책임은 무효를 주장하는 표의자에게 있다. (입증책임자 : 표의자)
3. 제3자에 대한 관계
	ㄴ 비진의표시가 예외적으로 무효가 되는 경우에도 '선의의 제3자'에게 대항하지 못한다.
	거래의 안전을 위한 규정으로 '등기의 공신력을 인정하지 않는' 민법 체계에서는 사실상 등기에 공신력을 부여하는 기능을 한다?
	ㄴ 보호되는 제 3자는 '선의이면 족하고', 무과실까지는 요구하지 않으며, 선의는 추정되므로 제3자가 악의라는 사실의 주장, 입증책임은 무효를 주장하는 자가 부담한다.
		ㄴ 선의는 모두 추정됨 -> 가만히 있기만 해도 선의라고 생각함 -> 상대방이 선의가 아님을 입증해야함
갑 -> 을 등기를 넘겨줌
	ㄴ 원칙 : 유효(을) -> 선의 and 무과실일 경우
	ㄴ 예외 : 무효(갑) -> 악의 or 과실 일 경우
	ㄴ 갑이 악의 or 과실임을 입증하지 못하면 등기는 을의 소유임
을 -> 병 병이 매입할 경우
	ㄴ 을이 권리자일 경우 -> 무조건 병이 취득하게 됨
	ㄴ 을이 무권리자일 경우 -> 원래는 취득할 수 없지만 .. -> 107조 2항 : 만약 병이 선의라면 취득하게 됨
		ㄴ 병이 취득하게 되는 것은 등기때문이 아닌 107조 2항의 법률에 따라서 취득한 것임
병의 취득조건
	ㄴ 을이 선의, 무과실 -> 을이 권리자 -> 병이 선, 악을 묻지 않고 취득
	ㄴ 을이 악의 혹은 과실 -> 갑이 권리자 -> 병이 선의인 경우라면 취득(과실, 무과실은 상관 없음)
		ㄴ 등기를 믿어서가 아닌 법률이 보호해준거임

법률 격언 : 무권리자가 권리를 내어줄 수 없음

통정허위표시
제108조 [통정한 허위의 의사표시]
1. 상대방과 통정한 허위의 의사표시는 무효로 한다.
2. 전항의 의사표시의 무효는 선의의 제3자에게 대항하지 못한다.

(1) 의의
통정 : 상대방과 짜고 함 -> 이러한 허위표시를 가장행위라고 한다.
	ㄴ ex) 친구랑 짜고 의사없이 소유권만 이전
(2) 요건
1. 의사표시가 있어야함
2. 의사 표시가 불일치하여야 함
3. 표의자가 의사와 표시의 불일치를 알아야 함 - 1,2,3까지가 비진의 ++아래 항목까지 더해지면 통정허위표시
4. 상대방과 통정(합의)하여야 함
5. 이유는 묻지 않음 ....

(3) 효과
1. 당사자 사이의 효력
- 당사자 사이에서는 언제나 무효
- 허위표시 자체가 불법은 아니므로 불법원인급여에 관한 규정은 적용되지 않는다.
- 이미 급부한 거은 반환 청구할 수 있다.
2. 제 3자에 대한 효력
- 선의의 제3자에게 대항할 수 없음
- 제 3자의 선의는 모두 추정됨(상대방쪽에서 입증을 해야만 함)
- 허위표시 이전에 법률관계를 맺은 자나 포괄승계인은 제3자가 될 수 없다.

승계 : 권리가 갑에게서 을로 넘어감
ㄴ 특정승계 : 정해진 것만 넘어감
	ㄴ 갑이 을에게 팜(매매, 증여)
ㄴ 포괄승계 : 불특정적으로 넘어감
	ㄴ 상속, 회사의 합병
	ㄴ 당사자의 지위로 인정되기 때문에 제3자로 보지 않음(병이 을의 상속인일 경우, 병을 을과 동일인물로 봄)
사례
갑 (부동산의 소유자) -> 을
A (갑의 채권자)
1. 갑이 A의 돈을 빌리고 안 갚음 -> A가 강제집행을 하려 함 -> 갑이 을(친구)에게 소유권을 넘김(가장 매매)
	ㄴ 비진의표시 <-> 통정허위표시 : 통정허위표시는 서로 의사와 표시가 불일치 된다는 것을 암
2. 가장매매가 사회질서위반은 아님 -> 불법원인급여의 적용은 없음 -> 부당이득반환을 요구할 수 있음
3. 을이 병에게 부동산을 팜 -> 병은 무권리자에게서 산 거임 -> 단 병이 선의라면 소유권을 취득할 수 있음
	ㄴ 갑이 병(선의의 제3자)에게 대항할 수 없기 때문에
	ㄴ 병은 선의로 추정됨, 무과실까지는 필요하지 않음(부주의했어도 상관없음)
	ㄴ 무효등기를 믿었기 때문이 아닌, 법률에 따라 보호받는 것임
